DYAC

Did You Ask Claude?

When you're stuck, this is step 1.
Paste the error. Describe what you want.

Prompting

Tips for Claude Code

  • Be specific about what you want
  • Give context (what you're building, why)
  • Paste errors directly back
  • Ask it to explain if confused
  • Start simple, iterate
Process

The Loop

1. Try something
2. Read the error
3. Paste to Claude
4. Repeat

This is 90% of the work.

Get Started

Setup

  • Get Claude Pro: claude.ai/upgrade ($20/mo)
  • Open Terminal (Cmd + Space, type "Terminal")
  • Install: npm install -g @anthropic-ai/claude-code
  • Create folder: mkdir my-project && cd my-project
  • Start: claude
